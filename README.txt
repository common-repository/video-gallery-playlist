== Installation ==

= AUTOMATIC INSTALLATION (EASIEST WAY) =

To do an automatic install of the plugin, log in to your WordPress dashboard, navigate to the Plugins menu, and
click Add New. In the search field type **"FancyTube - Video Gallery".** Once you have found it you can install it by simply
clicking "Install Now" and then "Activate". [__Watch an Instructions (How to) Video__](https://www.youtube.com/watch?v=N6GopKZiuGw).

= MANUAL INSTALLATION =

* Download the **FancyTube - Video Gallery** plugin
* Navigate to the **Add New** in the plugins dashboard
* Navigate to the **Upload** area
* Select the plugin from your computer
* Click **install Now**
* Activate the plugin in the Plugin dashboard
